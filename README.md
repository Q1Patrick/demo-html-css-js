# Run Locally

Clone the project:

```bash
git https://github.com/Q1Patrick/demo-html-css-js.git
```

Go to the project folder:

```bash
cd MyPortfolio
```

Install the **Live Server** extension in VSCode.

Open the project with Live Server:

- **Windows:** Press `ALT + L + O`
- **Mac:** Press `COMMAND + L + O`
